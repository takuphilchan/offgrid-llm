Quick Start

OffGrid LLM is an offline AI platform for edge computing. Select a model from the Chat tab and start asking questions.

Features

  • 100% offline inference
  • OpenAI-compatible API
  • GPU acceleration support
  • Multiple model support
  • Streaming responses

API Endpoints

GET /health

Check system health and resource usage

GET /v1/models

List all available models

POST /v1/chat/completions

Send chat completion requests

Example Request

curl -X POST http://localhost:11611/v1/chat/completions \ -H "Content-Type: application/json" \ -d '{ "model": "your-model-id", "messages": [ {"role": "user", "content": "Hello!"} ], "stream": false }'

System Requirements

  • 4GB+ RAM (8GB+ recommended)
  • 2GB+ disk space
  • Linux/Windows/macOS
  • Optional: NVIDIA GPU with CUDA

Model Format

Models must be in GGUF format. Place .gguf files in /var/lib/offgrid/models/

CLI Commands

offgrid serve

Start the server

offgrid list

List installed models

offgrid download <model>

Download a model

Environment Variables

  • PORT - Server port (default: 11611)
  • MODELS_DIR - Models directory
  • MAX_CONTEXT - Max context size
